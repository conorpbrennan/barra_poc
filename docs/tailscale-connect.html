Troubleshooting

The link won't load / “can't reach this site” / times out

If you're on a locked-down corporate network, its web filter may be blocking the address. First confirm the link works from a different network (e.g. your phone on cellular). If it loads there but not on the corporate network, your IT may need to allow risk.barra-poc.com — contact us (below) and we'll help with the request.

My username / password / token isn't accepted

They're case-sensitive and easy to mistype — copy and paste rather than typing if you can. If it still fails, ask your contact to re-send them.

I get a certificate warning

You shouldn't — these use a valid public HTTPS certificate. If you see one, you've likely typed http:// or a slightly wrong address; use the exact https://risk.barra-poc.com/… links above. The ++ in the dashboard address is intentional.

The page loads but shows “404”

The base address https://risk.barra-poc.com/ intentionally shows nothing — go straight to the full links above (ending in /flexagg++/ or /jupyter/).
